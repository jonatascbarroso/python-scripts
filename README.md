## [Base64 Decode](https://github.com/jonatascbarroso/python-scripts/tree/master/base64-decode)

This script decode a base64 file.

```
$ base64-decode.py

Base64 file to be decoded? encoded.pdf
Output file? decoded.pdf
Done
```
